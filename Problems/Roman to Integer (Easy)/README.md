Link: https://leetcode.com/problems/roman-to-integer/

Difficulty: Easy

Explanation:
    
    Given a roman numeral, convert it to an integer.

Example:
    
    Input: s = "LVIII"
    
    Output: 58
    
    Explanation: L = 50, V= 5, III = 3.

Table:

    Symbol       Value
    
    I             1
    
    V             5
    
    X             10
    
    L             50
    
    C             100
    
    D             500
    
    M             1000
